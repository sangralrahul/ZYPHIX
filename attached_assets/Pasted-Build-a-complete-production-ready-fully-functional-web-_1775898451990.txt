Build a complete, production-ready, fully functional web application called ZYPHIX — India's SuperLocal App. This is a full-stack consumer app similar to Flipkart/Blinkit but for Tier 2 India using kirana stores. Build it with React + Vite + TypeScript + Tailwind CSS. Every page must be fully built — no placeholders, no lorem ipsum, no "coming soon" content.

═══════════════════════════════════════════════
TECH STACK
═══════════════════════════════════════════════
- Frontend: React 18 + Vite + TypeScript
- Styling: Tailwind CSS with custom config
- Routing: React Router v6
- Auth: Firebase Authentication (Email/Password + Google OAuth)
- Database/State: Firebase Firestore + React Context
- Maps: Google Maps JavaScript API (@react-google-maps/api)
- Animations: Framer Motion
- Icons: Lucide React
- Notifications: react-hot-toast
- Forms: React Hook Form
- HTTP: Axios (for any external calls)
- Package manager: npm

Install all dependencies in package.json before building.
